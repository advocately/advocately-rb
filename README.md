# Advocately RubyGem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'advocately'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install advocately

## Usage

```
Advocately::Base.api_token = 'test'
Advocately::Base.app_id = '1'
Advocately::Review.all
```
